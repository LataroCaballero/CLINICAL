Internal state: type BannerPhase = 'visible' | 'classified' | 'gone'. useState for phase (starts 'visible'). useState for classifiedAs string (label shown in green check state). useRef for the setTimeout timer ref (typed as ReturnType of setTimeout or null).

Store selectors: const session = useLiveTurnoStore((s) => s.session); const bannerDismissed = useLiveTurnoStore((s) => s.bannerDismissed); const dismissBanner = useLiveTurnoStore((s) => s.dismissBanner).

Visibility gate — all hooks are declared first, then the early returns: if (!session || session.pacienteFlujo !== 'PENDIENTE') return null; if (bannerDismissed || phase === 'gone') return null. The equality check MUST be strict === 'PENDIENTE' — null and other values must not trigger the banner.

useEffect cleanup: useEffect with empty dependency array that returns a cleanup function calling clearTimeout(timerRef.current) if timerRef.current exists. This prevents the 2-second setTimeout from firing after unmount.

handleClassify function takes flujo as 'CIRUGIA' | 'TRATAMIENTO'. Sets classifiedAs to 'Cirugía' or 'Tratamiento'. Sets phase to 'classified' immediately. Calls api.patch to /pacientes/{session.pacienteId}/flujo with body {flujo} and catches errors silently (empty catch). Assigns timerRef.current = setTimeout calling dismissBanner() then setPhase('gone') after 2000ms.

handleDismiss function: calls dismissBanner() only. Does not call setPhase — the store's bannerDismissed becoming true will cause the visibility gate to return null on next render.

Render when phase === 'classified': a div styled bg-green-50 border-b border-green-200 text-green-800 flex items-center gap-2 px-4 py-2 text-sm. Inside: Check icon (w-4 h-4 shrink-0), span reading "Clasificado como {classifiedAs}".

Render when phase === 'visible': a div styled bg-amber-50 border-b border-amber-300 text-amber-800 flex items-center gap-3 px-4 py-2 text-sm. Inside: AlertTriangle icon (w-4 h-4 shrink-0), a span with flex-1 reading "Paciente sin clasificar — ¿Cirugía o Tratamiento?", a shadcn Button (size="sm" variant="outline", classes h-7 px-2 text-xs border-amber-400 text-amber-800 hover:bg-amber-100) calling handleClassify('CIRUGIA') labeled "Cirugía", another Button with same styling calling handleClassify('TRATAMIENTO') labeled "Tratamiento", and a plain HTML button (not shadcn) styled ml-1 p-1 rounded hover:bg-amber-100 with aria-label="Descartar" calling handleDismiss, containing the X icon (w-4 h-4).

Export the component as a named export: export function LiveTurnoFlujoBanner().
