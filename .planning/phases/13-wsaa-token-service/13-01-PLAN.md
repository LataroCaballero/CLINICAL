Then create `backend/src/modules/wsaa/wsaa.service.ts` (GREEN phase):
- `@Injectable()` class implementing `WsaaServiceInterface`
- Constructor injection: `@Inject(WSAA_REDIS_CLIENT) private readonly redis: Redis | null`, `PrismaService`, `EncryptionService`, `ConfigService`
- Private fields: `mutexMap = new Map<string, Mutex>()`, `logger = new Logger(WsaaService.name)`
- `private async loadConfig(profesionalId)`: queries `prisma.configuracionAFIP.findUniqueOrThrow({ where: { profesionalId } })`, decrypts cert+key via EncryptionService, returns `{ certPem, keyPem, cuit, ambiente }`
- `private buildTra(service)`: extracted from AfipConfigService (same implementation — TRA XML with 10min expiry window, `-03:00` timezone)
- `private signTra(traXml, certPem, keyPem)`: node-forge CMS signing:
  - `forge.pki.certificateFromPem(certPem)`, `forge.pki.privateKeyFromPem(keyPem)`
  - `forge.pkcs7.createSignedData()`, set `p7.content = forge.util.createBuffer(traXml, 'utf8')`
  - `p7.addCertificate(cert)`, `p7.addSigner({ key, certificate: cert, digestAlgorithm: forge.pki.oids.sha256, authenticatedAttributes: [contentType, messageDigest, signingTime] })`
  - `p7.sign({ detached: false })` — CRITICAL: must be false (AFIP requires content embedded)
  - Return `Buffer.from(forge.asn1.toDer(p7.toAsn1()).getBytes(), 'binary').toString('base64')`
- `private async callWsaa(certPem, keyPem, ambiente, service)`: builds TRA, signs, sends SOAP to WSAA URL (HOMOLOGACION → wsaahomo.afip.gov.ar, PRODUCCION → wsaa.afip.gov.ar), parses token/sign/expirationTime from XML response, returns AccessTicket. Timeout 15s. On network/5xx errors throw (do NOT catch — let caller handle retry).
- `private async redisSafeGet(key)`: try/catch Redis GET, return null on error + log warning
- `private async redisSafeSet(key, value, ttlSeconds)`: try/catch Redis SET with EX, void on error + log warning
- `async getTicket(profesionalId, service)`:
  1. loadConfig(profesionalId) to get cuit, certPem, keyPem, ambiente
  2. Acquire/create mutex for `${profesionalId}:${cuit}`
  3. Inside `mutex.runExclusive`: check Redis cache → if hit return parsed AccessTicket → if miss call WSAA → compute TTL (expiresAt - now - 5min in seconds) → if TTL > 0 store in Redis → return ticket
  4. Redis key: `afip_ta:${profesionalId}:${cuit}:${service}`
- `async getTicketTransient(certPem, keyPem, ambiente, service)`: calls `callWsaa` directly — no mutex, no Redis, no loadConfig

Run tests to GREEN. Refactor if needed.
