    Make the following four additions to `backend/src/prisma/schema.prisma`:

    1. **Add `precioBase` and `insumos` relation to the existing `Tratamiento` model** (after the `updatedAt` line):
    ```prisma
    precioBase  Decimal?           @db.Decimal(10, 2)
    insumos     TratamientoInsumo[]
    ```

    2. **Add new `TratamientoInsumo` model** (insert after the closing brace of `Tratamiento`):
    ```prisma
    model TratamientoInsumo {
      id            String      @id @default(uuid())
      tratamientoId String
      productoId    String
      cantidad      Decimal     @db.Decimal(10, 3)
      tratamiento   Tratamiento @relation(fields: [tratamientoId], references: [id], onDelete: Cascade)
      producto      Producto    @relation(fields: [productoId], references: [id])

      @@unique([tratamientoId, productoId])
      @@index([tratamientoId])
    }
    ```

    3. **Add new `CirugiaCatalogo` model** (insert after `TratamientoInsumo`):
    ```prisma
    model CirugiaCatalogo {
      id              String          @id @default(uuid())
      nombre          String
      precioARS       Decimal?        @db.Decimal(10, 2)
      precioUSD       Decimal?        @db.Decimal(10, 2)
      precioBase      Decimal?        @db.Decimal(10, 2)
      duracionMinutos Int?
      profesionalId   String
      activo          Boolean         @default(true)
      createdAt       DateTime        @default(now())
      updatedAt       DateTime        @updatedAt
      profesional     Profesional     @relation(fields: [profesionalId], references: [id])
      insumos         CirugiaInsumo[]

      @@unique([nombre, profesionalId])
      @@index([profesionalId, activo])
    }
    ```

    4. **Add new `CirugiaInsumo` model** (insert after `CirugiaCatalogo`):
    ```prisma
    model CirugiaInsumo {
      id         String          @id @default(uuid())
      cirugiaId  String
      productoId String
      cantidad   Decimal         @db.Decimal(10, 3)
      cirugia    CirugiaCatalogo @relation(fields: [cirugiaId], references: [id], onDelete: Cascade)
      producto   Producto        @relation(fields: [productoId], references: [id])

      @@unique([cirugiaId, productoId])
      @@index([cirugiaId])
    }
    ```

    5. **Add back-relations to `Producto` model** (find the `Producto` model and append inside it):
    ```prisma
    tratamientoInsumos TratamientoInsumo[]
    cirugiaInsumos     CirugiaInsumo[]
    ```

    6. **Add back-relation to `Profesional` model** (find the `Profesional` model and append inside it):
    ```prisma
    cirugiasCatalogo CirugiaCatalogo[]
    ```

    Do NOT modify any other models or fields. Do NOT change existing `Tratamiento` fields — only add the two new fields.
