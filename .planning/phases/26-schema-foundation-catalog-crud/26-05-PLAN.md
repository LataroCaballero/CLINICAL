    - `const [search, setSearch] = useState('')` — combobox search string

    **Data:** Use `useInventario(profesionalId)` to fetch the professional's stock. Filter locally: products whose `nombre` contains `search` (case-insensitive). Exclude products already in `insumos` state (by `productoId`).

    **Add insumo handler:**
    ```typescript
    const handleAdd = (inventarioItem: Inventario) => {
      const nuevo: InsumoLocal = {
        productoId: inventarioItem.producto.id,
        nombre: inventarioItem.producto.nombre,
        cantidad: 1,
        costoBase: inventarioItem.producto.costoBase !== undefined
          ? Number(inventarioItem.producto.costoBase)
          : null,
        unidadMedida: inventarioItem.producto.unidadMedida ?? null,
      };
      const updated = [...insumos, nuevo];
      setInsumos(updated);
      onChange(updated);
      setOpen(false);
      setSearch('');
    };
    ```

    **Update quantity handler:**
    ```typescript
    const handleCantidadChange = (productoId: string, value: string) => {
      const parsed = parseFloat(value);
      if (isNaN(parsed) || parsed <= 0) return;
      const updated = insumos.map(i =>
        i.productoId === productoId ? { ...i, cantidad: parsed } : i
      );
      setInsumos(updated);
      onChange(updated);
    };
    ```

    **Remove handler:**
    ```typescript
    const handleRemove = (productoId: string) => {
      const updated = insumos.filter(i => i.productoId !== productoId);
      setInsumos(updated);
      onChange(updated);
    };
    ```

    **Render structure:**
    1. Combobox section: `<Popover open={open} onOpenChange={setOpen}>` wrapping a trigger button ("+ Agregar insumo") and `<PopoverContent>` with `<Command>` containing `<CommandInput>`, `<CommandEmpty>No se encontraron productos</CommandEmpty>`, `<CommandGroup>` listing filtered inventory items as `<CommandItem>`.

    2. Table section (show only if `insumos.length > 0`): HTML `<table>` with `<thead>` (Producto | Cantidad | "") and `<tbody>` mapping over `insumos`. Each row: product name cell, `<Input type="number" min="0.001" step="0.001" value={insumo.cantidad}` cell (with `onChange` calling `handleCantidadChange`), and a `<Button variant="ghost" size="icon">×</Button>` cell calling `handleRemove`.

    Use Tailwind for styling: `text-sm`, `border rounded`, compact row padding `py-1 px-2`. Import `Input`, `Button` from `@/components/ui/`. No additional dependencies needed.

    Sync with `initialInsumos` prop changes using `useEffect`:
    ```typescript
    useEffect(() => {
      setInsumos(initialInsumos ?? []);
    }, [initialInsumos]);
    ```
    This ensures the component resets when the modal opens for a different item.

      const queryClient = useQueryClient();
      return useMutation({
