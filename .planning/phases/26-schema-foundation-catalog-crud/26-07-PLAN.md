  <action>
    Create `GestionCirugias.tsx` as a `"use client"` component mirroring `GestionTratamientos.tsx` structure but for cirugías. 

    **Props:**
    ```typescript
    interface GestionCirugiaProps {
      profesionalId?: string;
    }
    ```

    **State:**
    - `modalOpen: boolean` — create/edit dialog open
    - `editingCirugia: CirugiaCatalogo | null` — null = create mode
    - `form: { nombre: string; precioARS: string; precioUSD: string; duracionMinutos: string }` — string inputs for controlled form
    - `insumosLocal: InsumoLocal[]` — managed by InsumosEditor

    **Hooks to use:**
    - `useCirugiasCatalogo(profesionalId)` — list
    - `useCreateCirugiaCatalogo(profesionalId)` — create
    - `useUpdateCirugiaCatalogo(profesionalId)` — update
    - `useDeleteCirugiaCatalogo(profesionalId)` — soft delete
    - `useSetInsumosCirugia(profesionalId)` — save insumos batch
    - `useRecalcularPrecioCirugia(profesionalId)` — recalculate precioBase

    **Table columns:** Nombre | Precio ARS | Precio USD | Duración (min) | Costo insumos | Acciones

    - Precio ARS: `precioARS != null ? '$' + Number(precioARS).toLocaleString('es-AR', { minimumFractionDigits: 2 }) : '—'`
    - Precio USD: `precioUSD != null ? 'USD ' + Number(precioUSD).toLocaleString('en-US', { minimumFractionDigits: 2 }) : '—'`
    - Duración: `duracionMinutos != null ? duracionMinutos + ' min' : '—'`
    - Costo insumos: `precioBase != null ? '$' + Number(precioBase).toLocaleString('es-AR', { minimumFractionDigits: 2 }) : '—'`
    - Acciones: Edit button (pencil icon or "Editar" text) + Delete button with confirmation

    **Modal form fields:**
    - Nombre (required text input)
    - Precio ARS (optional number input, min 0, step 0.01)
    - Precio USD (optional number input, min 0, step 0.01)
    - Duración estimada (min) (optional integer input, min 1)
    - InsumosEditor section (inline, same pattern as GestionTratamientos)
    - "Recalcular desde insumos" button (disabled if no insumos or isPending)
    - Warning `<p className="text-xs text-amber-600">` if any insumo has `costoBase === null`

    **Open modal for create:** reset form to empty strings, `insumosLocal = []`, `editingCirugia = null`.

    **Open modal for edit:** populate form from `cirugia.*`, populate `insumosLocal` from `cirugia.insumos` mapping to `InsumoLocal` shape (same mapping as GestionTratamientos Plan 06).

    **Save handler:**
    1. Parse number fields: `parseFloat(form.precioARS) || undefined` (undefined if empty/invalid)
    2. If `editingCirugia`: call `updateMutation.mutateAsync({ id: editingCirugia.id, dto: { nombre, precioARS, precioUSD, duracionMinutos } })`
    3. If create: call `createMutation.mutateAsync({ nombre, precioARS, precioUSD, duracionMinutos })`
    4. After create/update succeeds, get the returned `id` and call `setInsumosMutation.mutateAsync({ id, insumos: insumosLocal.map(...) })`
    5. Close modal, reset state

    **Delete handler:** confirmation dialog (use `window.confirm` or a simple inline confirm button state pattern like GestionTratamientos). Calls `deleteMutation.mutateAsync(id)`.

    Style consistently with `GestionTratamientos` — use the same `Card`, `Button`, `Dialog`, `Table` shadcn/ui components.

    Show loading state from `isLoading` on the cirugías query while fetching.
