**Paso 5: Crear el handler de confirmacion**

Definir la función `handleConfirmSwitch` en el cuerpo del componente (antes del return):
```tsx
const hasDraftHC = !!(draftData.hcFormDraft && !draftData.hcFormDraft.saved);

const handleConfirmSwitch = async () => {
  if (!pendingTurnoId) return;
  try {
    // Paso 1: cerrar sesion actual SIN auto-guardar HC draft
    await cerrarSesion.mutateAsync({});
  } catch {
    // cerrarSesion.onError ya maneja el toast de error
    // No abrir el nuevo turno
    setShowSwitchDialog(false);
    return;
  }
  // Paso 2: abrir el nuevo turno (si cerrar-sesion ok)
  setShowSwitchDialog(false);
  iniciarSesion.mutate(pendingTurnoId);
  // Si iniciarSesion falla, su onError muestra toast separado
};
```

      >

      </AlertDialogAction>
    </AlertDialogFooter>
  </AlertDialogContent>
</AlertDialog>
