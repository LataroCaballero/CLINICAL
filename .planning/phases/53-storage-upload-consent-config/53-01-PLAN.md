</objective>

<interfaces>
<!-- Contract this plan defines; 53-02 (ConsentimientosService) consumes it. -->
StorageService (backend/src/modules/storage/storage.service.ts):
  - save(buffer: Buffer, profesionalId: string, originalExt?: string): Promise<string>
      → writes uploads/{profesionalId}/{randomUUID}.pdf, returns relativePath "{profesionalId}/{uuid}.pdf"
  - resolvePath(relativePath: string): string
      → absolute path under the uploads root; throws BadRequestException on traversal
  - getPublicUrl(relativePath: string): string
      → `${BACKEND_URL}/uploads/{relativePath}` (BACKEND_URL fallback http://localhost:3001 + warn)
  - readFile(relativePath: string): Promise<Buffer>
      → reads a stored file via resolvePath (used by UploadsController)
  - No delete method (history is preserved per D-05).
</interfaces>
</context>

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| unauthenticated client → /uploads/* | Public file serving with no auth; only the UUID path and attachment disposition protect the file |
| unauthenticated client → /presupuestos/public/* | Existing public endpoint, previously unthrottled |
| any client → entire API | No global guard existed before this plan |
| request param → filesystem path | profesionalId/filename params are resolved to a disk path |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-53-01 | Information Disclosure | UploadsController path resolution | mitigate | StorageService.resolvePath normalizes + rejects traversal (path.relative check, BadRequestException) before any fs read (D-10) |
| T-53-02 | Denial of Service | All API endpoints | mitigate | Global ThrottlerGuard via APP_GUARD, ttl 60000/limit 100 (D-07) → HTTP 429 |
| T-53-03 | Denial of Service | /presupuestos/public + /uploads | mitigate | Strict @Throttle tier (limit 20/min) on the unauthenticated presupuestos endpoint (D-08) |
| T-53-04 | Spoofing | /uploads/* (no auth) | accept | Public-by-design (D-09): security is the unguessable UUID filename + Content-Disposition: attachment; no PII in path, files are consent templates intended for patient download |
| T-53-05 | Information Disclosure | getPublicUrl when BACKEND_URL unset | accept | Falls back to localhost with a logged warning; production must set BACKEND_URL (documented in CLAUDE.md, existing convention) |
| T-53-SC | Tampering | @nestjs/throttler install | accept | Already present in backend/package.json (v6.4) — no new package install in this plan |
</threat_model>
