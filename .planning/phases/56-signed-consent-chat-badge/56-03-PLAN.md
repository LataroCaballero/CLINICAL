Implement a stamping method following RESEARCH Pattern 1 exactly: accept `{ templateBuffer, signaturePngBuffer, metadata: { fechaUtc, ip, userAgent, version } }`; `PDFDocument.load(templateBuffer)`; take the last page via `getPages()`; `embedFont(StandardFonts.Helvetica)` + `HelveticaBold` (no fontkit); `embedPng(signaturePngBuffer)`; draw a white `drawRectangle` FIRST to flatten PNG alpha (Pitfall F), then the forensic border box, then `drawImage` for the signature (bottom-right of box), then `drawText` lines for `CONSENTIMIENTO FIRMADO DIGITALMENTE`, fecha (UTC), IP, truncated userAgent, and `Versión del consentimiento`. Call `pdfDoc.save()` → wrap `Buffer.from(uint8)` → compute `crypto.createHash('sha256').update(pdfBuffer).digest('hex')` OVER THE FINAL BUFFER (Pitfall B / D-02) and return `{ pdfBuffer, hashSha256 }`. Do NOT print the hash inside the PDF (circular, D-02). Also expose a PNG magic-byte validator (compare first 8 bytes to `\x89PNG\r\n\x1a\n`) mirroring the existing `%PDF-` check style.
