## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| navegador del paciente (portal JWT) -> paciente-portal API | El paciente esta autenticado por JWT scope portal-paciente; no debe poder actuar sobre otro paciente |
| body del request -> escritura en BD | Campos extra en el body (indicacionesLeidas legacy, pacienteId inyectado) no deben afectar la escritura |
